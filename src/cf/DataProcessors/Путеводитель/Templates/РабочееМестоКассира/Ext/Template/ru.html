Рабочее место кассира

Рабочее место кассира демонстрируется в конфигурации в обработке РабочееМестоКассира.

Используйте пользователя Кассир для ознакомления с рабочем местом кассира. Для пользователя установлен режим основного окна клиентского приложения РабочееМесто, в котором окно клиентского приложения максимально облегченно с помощью скрытия большинства панелей.

В обработке содержатся следующие формы:

  • Форма настройки рабочего места кассира
    • Позволяет выбрать группу товаров отображающуюся на основной форме, а так же валюту используемую для отображения цен.
  • Форма оплаты наличными 
    • В которой реализован пример интерфейса для приема оплаты кассиром.
  • Основная форма 
    • Выводит список групп товаров и самих товаров в виде карточек с привычным для современных кассовых аппаратов и касс самообслуживания видом.
    • Для отображения на форме выбираются группы товаров с установленным реквизитом  ОтображатьВРабочемМестеКассира в значении Истина.
    • Для каждой группы товаром можно выбрать картинку и цвет фона. Заполнение реквизитов доступно на форме редактирования группы товара.
